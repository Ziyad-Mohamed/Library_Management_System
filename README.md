# 📚 Library Management System (Dart OOP Project)

A simple and clean Library Management System built using Dart and Object-Oriented Programming (OOP).  
The system manages books and users through a console-based application.

---

## 🚀 Features

- Add new books  
- Remove books by ID  
- View all books  
- Search for books by title  
- Borrow books with user validation  
- User system for borrowing process  

---

## 🧠 Concepts Used

- Object-Oriented Programming (OOP)  
- Classes & Objects  
- Encapsulation  
- Lists  
- Methods & Functions  
- System design basics  

---

## 🏗️ Project Structure

main.dart → Entry point of the application  
Book.dart → Book model class  
BookManager.dart → Handles all book operations  
User.dart → User model class  

---

## ⚙️ How It Works

- Books are created using the Book class  
- Users are created using the User class  
- BookManager controls all operations:
  - Add books  
  - Remove books  
  - Search books  
  - Borrow books  
- main.dart runs and tests the system  

---
